# 🚀 Section 01: Navbar & Header Components

This section contains clean layouts for modern website navigation bars, logos, and search headers.

## 📂 Sub-Folders & Projects
* 📁 **[html-css-variants](./html-css-variants/)** — Static header architectures and structural menu alignments using HTML and CSS.
* 📁 **[html-css-js-variants](./html-css-js-variants/)** — Interactive navigation bars featuring mobile hamburger toggles, sticky scroll locks, and active link states.

## 🧠 Core Learning Objectives
* Structuring accessible navigation anchors with proper list weights.
* Aligning standard search fields alongside logo blocks.
